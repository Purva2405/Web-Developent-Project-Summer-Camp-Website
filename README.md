A full-stack **Summer Camp Website** developed using **HTML, CSS, JavaScript, and PHP**, running on a **XAMPP local server** with **MySQL (phpMyAdmin)** as the database. The project allows users to explore different types of camps, register/login, make payments, and submit feedback.


## Project Description

This website is designed to manage **summer camp activities for children**, such as:

* Sports camps
* Cooking camps
* Outdoor games camps

Users can choose any camp, register or log in, make a payment for the selected camp, and submit feedback. All user data, registrations, and payment details are stored in the **MySQL database using phpMyAdmin**.
